# Timeki Marketing Website

Marketing website for Timeki - GPS-verified time tracking for field service teams.

## Structure
- Root: HTML pages (index, privacy, terms, help)
- /assets/images: Logo, screenshots, icons
- /assets/css: Stylesheets (if needed)
- /assets/js: Scripts (if needed)

## Deployment
Deployed via Cloudflare Pages to timeki.app

## Note
The web app at timeki.app/app is hosted separately on Firebase.
